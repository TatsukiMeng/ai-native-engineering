# 编辑器规范化 Pass

当 JSON 场景的结构已经正确，但连接线路径仍然不自然时，使用这个 pass。

## 什么时候做

- binding 已经正确，但箭头没有从最合适的边离开节点
- 交叉在技术上成立，但视觉上很别扭
- 这张图需要借助 Excalidraw 拖动时的实时重算能力

## 在编辑器里怎么做

1. 在 Excalidraw 中打开 `.excalidraw` 文件。
2. 一次只拖一个节点，让编辑器先把相关箭头重算稳定。
3. 优先做小幅校正，不要一上来就重画整张图。
4. 如果某根连接还是很别扭，优先重新连接，而不是继续过拟合 JSON `points`。
5. 保存规范化后的 `.excalidraw`，再用项目渲染器重新导出 PNG。

## 心态

editor pass 不是权宜之计，而是当 JSON 级控制过粗时，专门用来收敛连接几何的规范化步骤。
