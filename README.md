# 🤖 AI 原生工程：Vibe Coding 进阶与全域自动化落地

欢迎来到本书的开源协作仓库！这是一本致力于平滑学习曲线、从“入门实战避坑”逐渐过渡到“高级全域自动化集群（Swarm）”的技术专著。

## 🎯 核心目标 (Objective)

打破“AI 只是高级自动补全”的旧认知，建立 Context（上下文）驱动的新心智，完成开发者的思想冷启动。不仅探讨 Vibe Coding 的概念，更将其落地为可复用、可度量的自动化工程实践。

## 🧠 核心理念 (Core Philosophy)

1. **融合视角**：既要有工业级的反常识痛点剖析（如：调试时间激增、逻辑与测试的自洽陷阱、模板化提效），也要有最前沿的自动化极客玩法（MCP, Playwright, Swarm）。
2. **渐进式认知**：前中期的章节聚焦于个人开发者的 IDE 提效、PostgreSQL 后端开发、Tauri/Wujie 前端跨端实战；后期的章节拔高到架构师视角，解决 Token 失控、集群编排和全域接管问题。
3. **极客与实战驱动**："Show, Don't Just Tell"。强调痛点与解法绑定，对每一种先进的 AI 解决方案，都必须指出容易踩的坑并给出工程化兜底方案。

## 📖 目录大纲速览 (Outline Preview)

本书共分五大篇章，循序渐进地带你走进 AI 原生工程的深水区：

- **第一篇：破局 —— 编程范式的第三次跨越**
  *认知与工具生态：从“你写 AI 补”到“你说 AI 做”的范式转移，深入解析 Cursor / Windsurf / MCP 等生态位。*
- **第二篇：暗礁 —— 真实场景的痛点与防线**
  *工业级避坑指南：直面黑盒调试、自我拟合（自洽陷阱）等 AI 辅助编程中的致命痛点，构建企业级防御体系。*
- **第三篇：实战 —— 落地全栈 AI 工作流**
  *核心工程重构：从零跑通基于 PostgreSQL, Tauri/Wujie 的全栈项目，体验 Ticket-based 拆解下的 AI 降维打击。*
- **第四篇：越界 —— 赋予 AI “手”与“眼”**
  *工具链自动化：借助 MCP 革命接管浏览器、终端、多媒体等十大工程维度的系统级应用图谱。*
- **第五篇：终局 —— 拥抱 Agent Swarm 与反脆弱设计**
  *架构师的进化：搭建虚拟研发团队（PM, QA 智能体），解决无限 Ping-Pong 效应与幻觉放大的终极反脆弱重构。*
